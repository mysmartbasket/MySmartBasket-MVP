# MySmartBasket

MySmartBasket is an intelligent grocery shopping application designed to help users
optimize their shopping basket through personalization, data analysis, and smart
recommendations.

## Project Status
This repository contains a stable and presentable version of the application.
Active development is carried out in a separate development branch.
